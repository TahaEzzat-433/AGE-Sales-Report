# AGE-Sales-Report
# Description
This project is an interactive Sales & Marketing Report that provides insights into a company's performance. It covers executive summary, product analysis, customer and supplier details, and visualizes key metrics to help make data-driven decisions.
# Features
**Executive Summary**: Overview of total revenue, number of orders, on-time delivery rate, average order value, and top-selling categories.

**Product Analysis**: Insights on stock availability, top products by quantity and revenue, and average order quantities.

**Customer Behavior Analysis** involves examining how customers interact with the company, identifying patterns in their behavior, and measuring the **retention rate** to understand customer loyalty and engagement.

**Supplier & Inventory Analysis Dashboard**This specialized dashboard provides critical insights into supply chain health and supplier performance, ensuring optimal stock levels and procurement efficiency.

**Interactive Filtering**: Filter by date, shipping method, order status,customer name,supplier name, and employees for dynamic analysis.

# Technologies Used
Data visualization tools (Excel)

Excel / CSV data processing

Interactive charts and graphs
# How to Run
1. Download or clone the repository.

2. Open the Excel file using Microsoft Excel.

3. Follow the instructions inside the file to view the reports.
